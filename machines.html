Parc existant

Nous disposons d'un parc de machines répartis sur les différents lieux que nous animons. Vous pouvez en réserver une directement sur le fabmanager.


Les étapes

Pour réserver une machine, il vous faudra:

  1. Adhérer
  2. Suivre une formation
  3. Réserver un créneau machine sur le fabmanager
  4. Utiliser !
